# Mohammed Abed - Personal Portfolio

A modern, single-page personal portfolio website built with Next.js 15 (App Router), TypeScript, Tailwind CSS v4, and Framer Motion.

## Features

-   **Modern Tech Stack**: Next.js 15, React 19, TypeScript, Tailwind CSS v4.
-   **Responsive Design**: Fully responsive for Mobile, Tablet, and Desktop.
-   **Animations**: Smooth scroll and reveal animations using Framer Motion.
-   **Centralized Data**: All content is stored in `src/data/portfolioData.ts` for easy editing.
-   **Performance**: Optimized for speed and SEO.

## Getting Started

### Prerequisites

-   Node.js 18.17 or later
-   npm

### Installation

1.  Clone the repository (if applicable) or navigate to the project directory.
2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Run the development server:

    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

### Content

All personal information, skills, projects, experience, and education data are stored in `src/data/portfolioData.ts`. Edit this file to update your portfolio content without touching the UI code.

### Styling

The project uses Tailwind CSS v4. Global styles and theme configuration are in `src/app/globals.css`.
You can customize colors, fonts, and other design tokens there.
